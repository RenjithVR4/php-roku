Roku PHP Library
================================================

PHP Library for communication with Roku External Control Protocol

## Installation

Install using composer

```
{
    "require" : "svilborg/php-roku"
}
```

## Requirements

* PHP Version >=5.3.2.
* PHP Httpful Library

## Running the tests

First, install PHPUnit with `composer.phar install --dev`, then run
`./vendor/bin/phpunit`.
